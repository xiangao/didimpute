# didimpute: Difference-in-Differences Imputation Estimator (Borusyak, Jaravel & Spiess)

A faithful R port of the Python did_imputation package implementing the
imputation estimator of Borusyak, Jaravel and Spiess (2024) for
staggered difference-in-differences, with event-study aggregation and
plotting.
